# windows平台emacs配置工程

## 1、如何使用？
下载emacs的windows版本，然后下载本工程，将工程所有目录、文件拷贝到emacs安装目录。
并将emacs目录放到D盘根目录（因为配置文件约定是在D盘，如是其它位置，请自行修改）。
最后将工程目录的.emacs拷贝到C盘根目录。
双击bin目录的runemacs.exe即可使用。

emacs下载地址：
http://ftp.gnu.org/gnu/emacs/windows/


## 2、这个emacs配置有什么功能？

** 支持yaml文件（后缀名为yml或yaml）
