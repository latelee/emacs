# emacs使用笔记 （Late Lee<li@latelee.org>）

声明：本笔记经过实践总结，但还是不敢肯定适合所有场景――但在本工程配置下是OK的。

标记说明：
c表示Ctrl键，m表示Alt键。
c-c c-f表示按住Ctrl键，同时按字母c键，松手，然后按住Ctrl键，同时按字母f键。（有点绕，实际操作是很快的）
如果没有“c”或“m”，表示只是按对应的字母键

## 文件操作类
c-x c-f   在emacs中打开文件，这时会提示要打开的文件的路径，可以使用Tab键补全文件名，打开后会覆盖当前那个窗口（可以打开多个窗口，再打开文件）
c-x c-s：保存文件
c-x c-c：  退出emacs（未保存的话会出现提示）

## 编辑操作类
m-u： 光标所在字符以及后面所有字符全部变成大写
m-l： 同上，变成小写 （注意，是字母“L”的小写，不是“i”的大写）
c-k：删除光标所在字符，一直到行尾（实际是剪切到缓冲区了，用c-y可以粘贴回来的）
c-d：删除点下的字符
m-d：删除点所在的单词

c-@：开始标记
c-W: 剪切（大写的“W”）
m-W：复制（大写的“W”）
c-y：粘贴

c-f: 点前移一格
c-b: 点后退一格
c-p: 点向上一行
c-n: 点向下一行
m-f: 点向前到下一个单词
m-b: 点移动到上一个单词
c-a: 点移动到行首
c-e: 点移动到行尾
m-a: 点移动到句首
m-e: 点移动到句尾
m-{: 点移动到段首
m-}: 点移动到段尾

m-g g (再输入行号): 跳到某一行
c-x [: 移至顶
c-x ]: 移至底

c-s: 向前搜索    
c-r: 向后搜索

c-g: 退出某些命令（如，按错了某些命令，可以用这个命令取消它）
c-x u：恢复上一个操作

好像不能直接使用Tab键的，如要使用，可以：c-q Tab

## 窗口类
c-x 2 (水平) c-x 3 (垂直):
将本窗口分割成多个窗口――是本窗口的内容被分割！故内容是一样的，可以在选中的窗口的再打开文件、编辑文件，等
c-x 0(数字0): 关闭本窗口――如果当前只打开一个窗口，输入此命令出现提示，大意是说你尝试关闭唯一的窗口(更准确的说法是buffer)。
c-x ^: 使窗口增高
c-x }: 使窗口增宽
c-x {: 使窗口变窄
c-x >: 窗口显示右边的内容
c-x <: 窗口显示左边的内容
c-x o: 切换到另一个窗(这是字母o)――跳到指定的还不知道，所以只能一个一个地跳了
c-x c-b: 显示当前的emacs打开了多少个buffer（列表有：在缓冲区的文件（或shell），大小，模式、位置）
c-x b: 选择某一个buffer，这时会提示选择哪一个
c-x k: 关闭某一个buffer，这时会显示默认要删除的文件，回车即删除默认的文件，如果要删除其它的，一定要输入列表中显示的全部名称，如，打开一个shell，要结束它就要输入：c-x k *shell*，这样才行
